spec-version:set-spec-properties

Full name:

org.glassfish.build:spec-version-maven-plugin:2.1-SNAPSHOT:set-spec-properties

Description:

Generate spec properties.

Attributes:

  • Requires a Maven project to be executed.
  • The goal is not marked as thread-safe and thus does not support parallel builds.
  • Binds by default to the lifecycle phase: validate.

Required Parameters

Name Type Since Description
<spec> Spec - The spec.
User property is: spec.

Optional Parameters

Name Type Since Description
<specMode> String - Mode. Allowed values are "javaee", "jakarta"
Default value is: jakarta.
User property is: specMode.

Parameter Details

<spec>

The spec.
  • Type: org.glassfish.spec.Spec
  • Required: Yes
  • User Property: spec

<specMode>

Mode. Allowed values are "javaee", "jakarta"
  • Type: java.lang.String
  • Required: No
  • User Property: specMode
  • Default: jakarta
