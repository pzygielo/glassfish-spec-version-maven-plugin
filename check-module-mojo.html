spec-version:check-module

Full name:

org.glassfish.build:spec-version-maven-plugin:2.1-SNAPSHOT:check-module

Description:

Maven Goal to enforce spec rules and fail the build.

Attributes:

  • Requires a Maven project to be executed.
  • The goal is not marked as thread-safe and thus does not support parallel builds.
  • Binds by default to the lifecycle phase: package.

Required Parameters

Name Type Since Description
<spec> Spec - Spec.
User property is: spec.

Optional Parameters

Name Type Since Description
<ignoreErrors> boolean - Ignore failures.
Default value is: false.
User property is: ignoreErrors.
<module> File - Module to verify.
User property is: module.
<specMode> String - Mode. Allowed values are "javaee", "jakarta"
Default value is: jakarta.
User property is: specMode.
