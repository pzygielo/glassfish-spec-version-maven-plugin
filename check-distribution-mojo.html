spec-version:check-distribution

Full name:

org.glassfish.build:spec-version-maven-plugin:2.1-SNAPSHOT:check-distribution

Description:

Check a set of spec artifact in a staging directory.

Attributes:

  • Requires a Maven project to be executed.
  • The goal is not marked as thread-safe and thus does not support parallel builds.
  • Binds by default to the lifecycle phase: package.

Required Parameters

Name Type Since Description
<dir> File - The directory containing the spec artifacts to process.
User property is: dir.
<specs> List - The specification configurations.
User property is: specs.

Optional Parameters

Name Type Since Description
<excludes> String - Exclude pattern.
User property is: excludes.
<includes> String - Include pattern.
Default value is: javax*.jar.
User property is: includes.

Parameter Details

<dir>

The directory containing the spec artifacts to process.
  • Type: java.io.File
  • Required: Yes
  • User Property: dir

<excludes>

Exclude pattern.
  • Type: java.lang.String
  • Required: No
  • User Property: excludes

<includes>

Include pattern.
  • Type: java.lang.String
  • Required: No
  • User Property: includes
  • Default: javax*.jar

<specs>

The specification configurations.
  • Type: java.util.List
  • Required: Yes
  • User Property: specs
